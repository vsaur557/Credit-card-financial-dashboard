# Credit-card-financial-dashboard
## 📌 Project Overview

This project analyzes credit card customer and transaction data using **SQL and Power BI**.
SQL was used for data extraction, cleaning, and transformation, while Power BI was used for data visualization and dashboard creation.

## 🛠 Tools & Technologies Used

* 🗄 SQL (PostgreSQL)
* 📊 Power BI
* 📈 Data Analysis
* 📉 Data Visualization
* 
## 🔄 Project Workflow
1. Data collected and stored in SQL database
2. Data cleaning and transformation using SQL queries
3. Data imported into Power BI
4. Data modeling and relationships created
5. Dashboard and visualizations built in Power BI
6.
